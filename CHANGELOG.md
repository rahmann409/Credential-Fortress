# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-16

### Added
- Complete educational password assessment toolkit.
- Strong safeguards rejecting system paths like `/etc/shadow`.
- `--confirm-lab` flow for first-time use.
- Interactive terminal UI using `rich` and `prompt_toolkit`.
- Plugin architecture for new hash algorithms and mutations.
- CI pipeline, Dockerfile, and reproducible environments.
- Comprehensive documentation, threat model, and lab exercises.
